# 📦 Lost & Found Tracker

A web application that helps users **report, track, and manage lost or found items**.
Users can submit items they lost or found, view their queries, mark them as resolved, and delete them when no longer needed.

This project demonstrates **React component-based architecture, state management, and UI interactions**.

---

## 🚀 Features

- ➕ **Add Lost/Found Items**
  Users can submit details about items they lost or found.

- 📋 **View Queries**
  Items are displayed in a clean **card-based UI**.

- 👤 **My Queries Page**
  Shows all the queries posted by the user.

- ✅ **Resolve Item**
  Mark items as resolved when they are found.

- ❌ **Delete Query**
  Remove items that are no longer relevant.

- 🎨 **UI Animations**
  Smooth animations when resolving or deleting a query.

- 💾 **Local Storage Support**
  Queries remain saved even after refreshing the page.
